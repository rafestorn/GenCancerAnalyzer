GenCancerAnalyzerAPI

This is a RESTful API for GenCancerAnalyzer where you can query the results of the analisis.

Try out the endpoints with SWAGGER or REDOC

Endpoints

Here is a summary of the Endpoints of the API:

/api/StudyCase

Get a list of studyCase.

  • id: Filter by id
  • project: Filter by project
  • data_type: Filter by datatype ('RNAseq' or 'miRNAs')
  • state: Filter by state ('DONE' or 'ANALISIS')
  • page: Page number
  • maxItems: Number of results per page
/api/StudyCase/{id}

Get a Study Case by id

/api/StudyCase/{id}/metadata

Get the metadata of the specified studyCase.

  • page: Page number
  • maxItems: Number of results per page
/api/StudyCase/{id}/rnaExpression

Get the voom normalized values of genes of the specified studyCase.

  • genes_ids: List of comma-separated genes_ids to filter
  • page: Page number
  • maxItems: Number of results per page
/api/StudyCase/{id}/rnaExpression/{gene_id}

Get the voom normalized values of the specified gene and studyCase.

  • sep: List of comma-separated mirnas_ids to filter
  • show_metadata: Set true to expand the JSON with the metadata info.
  • calculate_kmsurvival_function: Return km survival function
  • sample_type: Filter samples by the specified sample type
/api/StudyCase/{id}/differentialExpression

Get the differential expression values of the specified studyCase.

  • order_by: Attribute by which the results are sorted
  • group: Filter by group
  • page: Page number
  • maxItems: Number of results per page
/api/StudyCase/{id}/enrichAnalysis

Get the enrichment analysis results of the specified studyCase.

  • order_by: Attribute by which the results are sorted
  • category: Filter by category
  • page: Page number
  • maxItems: Number of results per page
/api/StudyCase/{id}/survivalAnalysis

Get the survival analysis results of the specified studyCase.

  • order_by: Attribute by which the results are sorted
  • page: Page number
  • maxItems: Number of results per page
