GenCancerAnalyzer

The application offers an APIRest to query the results of the analysis.

How Is the Analysis Performed?

GenCancerAnalyzer utilizes the "gdcRNATools" library, an R/Bioconductor package, to perform the majority of the analyses. Users can select the GDC project and the type of gene expression (RNA or miRNAs) to analyze.

Features:

  • Analize Projects: By selecting the project and data type get the metadata of the project and results from the differential expression analysis, enrichmente analysis (Only for RNA type), and univariate survial analysis. Also you can consult the results from analisis already made from other users.
  • Download Results: Download the results of the analisis on csv or as an excel file.
  • Plots: Visualize the results as plots to get better understanding of the ingormation.
  • Access the API: Discover the GenCancerAnalyzerAPI to get to the stored data from the analysis.
