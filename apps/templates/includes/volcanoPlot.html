A volcano plot is a graphical representation where:

  • Horizontal Axis: represents the effect size or fold change of a parameter. In this case the log2(fold change), which measures how much a particular gene changes between two conditions (PrimaryTumor-SolidTissueNormal). Positive values on the right side of the axis indicate upregulation, while negative values on the left side indicate downregulation.
  • Vertical Axis: represents the statistical significance of the parameter, expressed as the negative logarithm (base 10) of the p-value. The higher the values are on the y-axis means the lower are th p-values, therefore more significance.
  • Points: represent the genes. The color of the points indicates the significance of the parameter given the thresholds.
  • Thresholds: are the values that the user can set to filter the genes. The user can set the threshold for the p-value and the log2(fold change).
